Shipping Charge Estimator

A backend service that calculates shipping costs for B2B marketplace deliveries

Service Running • Local

What This Service Does

This service helps sellers ship goods to kirana stores across India. It automatically calculates shipping costs by considering the distance between warehouse and destination, product weight, and delivery speed preference. The system intelligently selects the most suitable transport mode for each shipment.

Use Cases

  • Calculate shipping charges between warehouse and customer location
  • Automatically select optimal transport mode based on distance
  • Support both standard and express delivery options
  • Find the nearest warehouse for a seller's location
  • Get fast responses through in-memory caching

API Quick Reference

Method Endpoint Description
GET /api/v1/shipping-charge Get shipping charge options
POST /api/v1/shipping-charge/calculate Calculate shipping cost
